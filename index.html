A multilayered set of security features at the hardware, operating system (OS), and system level to help defend against the sophisticated attacks of today and tomorrow. AMD Ryzen™ Series processors come integrated with a Microsoft Pluton security processor, a chip-to-cloud security technology designed and updated by Microsoft, that helps enhance the protection of Windows 11 PCs with continuous protection for user credentials, identities, personal data, and encryption keys3,4,5.

Did you know a PC in sleep mode is a data risk?

Help secure your entire system memory with AMD Memory Guard. AMD processors with PRO technologies are the world’s only processor family with AMD’s full memory encryption as a standard feature10. AMD Memory Guard helps protect data with full memory encryption should your PC be lost or stolen.
